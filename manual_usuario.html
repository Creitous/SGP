Término

Descripción

Aplicación web

Es un sistema anclada a la web que funciona gracias a la integración de servidores que aloja contenido de la página web.

Inicio de sesión

Proceso mediante el cual se permite controlar el acceso a la plataforma y evitar posibles danos.

Subir archivos

Función donde permite alojar archivos en un servidor para su descarga o visualización de su existencia en la pagina web.
